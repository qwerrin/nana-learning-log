## 🟢 すぐやりたい(効果大・難易度低)

- [ ] **X投稿の予約投稿**
  - 候補ツール:X Pro(旧TweetDeck)、Buffer、Hootsuite
  - 効果:毎日朝晩の張り付き不要
  - メモ:無料プランで十分か確認

- [ ] **Zenn → X 告知の自動化**
  - 候補ツール:Zapier / Make + Zenn RSS
  - 効果:Zenn公開と同時にXに告知が流れる
  - メモ:RSSフィードを使うのが定番パターン

- [ ] **週次ログのテンプレ自動生成**
  - 候補ツール:GitHub Actions
  - 効果:毎週月曜に自動でWXXファイルが作られる
  - メモ:cronスケジュール + create-file action

- [ ] **学習時間の自動計測**
  - 候補ツール:WakaTime(VS Code拡張)、Toggl
  - 効果:月次振り返りの時間集計が自動化
  - メモ:WakaTimeは無料プランあり

## 📝 自動化を検討する前のチェックリスト

「これ自動化したい!」と思ったとき、着手前に以下を確認:

- [ ] 本当に時間を奪っているか?(週○分以上)
- [ ] 自動化の構築時間と、その後の節約時間は釣り合うか?
- [ ] そのツールの月額コストは妥当か?
- [ ] 失敗したときのリカバリ手段はあるか?
