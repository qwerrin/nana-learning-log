## 📝 7つの投稿テンプレ（AIインプット対応）

### テンプレ1:今日の積み上げ(平日朝/夜)
```text
[Day XX] 今日の積み上げ

✅ [達成1]
✅ [達成2]
✅ [達成3]

明日は [明日やること]。

#今日の積み上げ #駆け出しエンジニア

**コツ**:
- 3つに絞る(多いと読まれない)
- 明日の予告で締める(継続感)
- ハッシュタグ2〜3個

---

### テンプレ2:学んだことシェア(週2回)

```
[技術名]の [概念A] と [概念B] って、
同じものだと思ってた…

正解:
🔹 [概念A] = [説明]
🔹 [概念B] = [説明]

[初学者目線での一言コメント、または感動したポイント]

#[技術タグ] #初心者
```

**コツ**:
- 「最初は誤解してた」→「正解はこう」の構造が刺さる
- 絵文字や記号で視覚的に整理
- 学習者の共感を呼ぶ

---

### テンプレ3:コードスクショ(週1〜2回)

```
[実装したこと]、ついに動いた！🌓

(ここにコードのスクショ画像を添付)

たった [コード行数] 行だけど、
[AI（Claudeなど）に頼んで工夫したポイントや、自力で頑張った点]

#Web開発 #[技術タグ]
```

**コツ**:
- VS Codeのテーマは「One Dark Pro」「Dracula」が映える
- スクショはコード部分だけ切り取る
- コード行数を強調(短さアピール)

---

### テンプレ4:ハマったこと→解決(週1回)

```
[30分など] ハマった話🫠

[何をしようとして]
[どう画面が崩れた/どんなエラーが出たか]

調べたら（またはAIに聞いたら）…
正解はこうでした：
[正解・正しい書き方]

エラーを恐れず、解決のプロセスを楽しむ！

#今日の積み上げ #プログラミング初心者
```

**コツ**:
- 「初心者あるある」が一番バズる
- 失敗を恥じない・むしろ堂々と
- 数字(30分など)を入れると具体性アップ
- これはZenn記事のネタにもなる

---

### テンプレ5:Zenn記事公開告知(土曜の17-21時)📰

```
今週の学習テーマ【[テーマ名]】の記録、Zennに書きました📝

[記事タイトル]
[Zenn URL]

「[読者に最も刺さる、この記事だけのオリジナルの見どころやTips]」

#Zenn #駆け出しエンジニア #[関連タグ]
```

**コツ**:
- タイトル → URL → 1行紹介 の流れ
- 公開後30分以内に投稿
- 金曜夜〜日曜朝が読まれやすい
- **これが週で最も重要な投稿**

### テンプレ6:読んだ記事の感想(週1回)

```
これ最高に勉強になった📚

### テンプレ7:週次振り返り(日曜夜)📊

```
今週の振り返り📊 [積み上げX週目]🌱

✅ [今週の1番の成果、またはモチベーションの変化]
✅ [Zenn公開記事や作った成果物の報告]
✅ [学習時間や、AIとタッグを組んで進めた実感]

来週やること:
🔹 [予定1]
🔹 [予定2]

一歩ずつ着実に！

#今日の積み上げ #駆け出しエンジニア
```

**コツ**:
- 日曜夜投稿が最も読まれる
- 達成済みと予定の両方を見せる
- 週次ログ(weekly-log/)とリンクさせると楽

## 🚫 やらないこと(ルール)

| NG | 理由 |
|---|---|
| ❌ 政治・宗教・年収マウントの投稿 | 燃えやすい・信頼を失う |
| ❌ 他社プロダクトのネガキャン | 長期的に信頼を失う |
| ❌ 完璧な投稿を目指す | 投稿できなくなる |
| ❌ 数字(いいね・フォロワー)に一喜一憂 | 心が疲れる |
| ❌ 1日3投稿以上の連投 | 質が落ちる、相手のタイムラインを圧迫 |
| ❌ AIで作った文章をそのままコピペ | 一次情報の価値がゼロになる |
